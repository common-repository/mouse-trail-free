== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/mouse-trail-free/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress.
3. Configure your settings under **Settings > Mouse Trail Options**.

== Frequently Asked Questions ==
= How do I enable the mouse trail? =
After activating the plugin, go to **Settings > Mouse Trail Options** and enable it.

= Can I change the color of the mouse trail in the free version? =
Yes, you can choose a basic color for the trail in the free version.

= What additional features does the premium version offer? =
The premium version offers full customization including control over the thickness, length, opacity, speed, and the ability to add gradient effects.
